Featured Project

Plant Doctor AI

A comprehensive deep learning-powered web application for plant disease detection and diagnosis

🎯 Project Overview

Plant Doctor AI is a sophisticated web application that leverages ResNet18 deep learning architecture to identify plant diseases from leaf images with high accuracy.

🔬 Technology Stack

Built using PyTorch for deep learning, Flask for backend, and modern HTML/CSS/JS for frontend. Trained on PlantVillage and PlantDoc datasets.

✨ Key Features

AI-powered predictions, Grad-CAM heatmap visualization, disease treatment recommendations, comprehensive disease database, and real-time analysis.

🌐 Deployment

Successfully deployed on Render with custom domain (plantdrai.xyz), featuring responsive design and optimized performance.

📊 Model Performance

Trained ResNet18 model capable of identifying 38+ plant disease classes with high confidence scores and visual explanations.

🎓 Impact

Provides farmers and gardeners with instant, accurate disease diagnosis, helping reduce crop losses and improve agricultural practices.
